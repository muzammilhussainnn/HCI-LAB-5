## 1. Goal
Splitting Lab 04 into three files to practice **Separation of Concerns**.

## 2. File Structure
* **`index.html`**: Defines the page structure.
* **`style.css`**: Handles all visual design and colors.
* **`script.js`**: Controls the interaction logic and click limits.

## 3. Why Split?
* **Clean Code:** Makes the project easier to read and manage.
* **Independence:** Styles or logic can be updated without breaking the HTML.
* **Efficiency:** Encourages organized coding habits.

## 4. Design Results
The interface still uses **Affordance (Blue)**, **Feedback (Yellow)**, and **Constraints (Red)** to guide the user, but the code is now modular.
